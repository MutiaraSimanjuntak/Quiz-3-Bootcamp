# 🧪 OrangeHRM Login Automation (Cypress)

## 📌 Deskripsi

Project ini berisi automation testing untuk fitur **Login** pada OrangeHRM menggunakan Cypress.
🔗 [https://opensource-demo.orangehrmlive.com/](https://opensource-demo.orangehrmlive.com/)

---

## ⚙️ Tools

* Cypress
* Visual Studio Code
* NodeJS

---

## ▶️ Cara Menjalankan

```bash
npm install
npx cypress open
```

Pilih file: `login.cy.js`

---

## ✅ Test Case

1. Login valid
2. Password salah
3. Username salah
4. Tanpa username
5. Tanpa password
6. Tanpa keduanya
7. Case sensitive password
8. Input dengan spasi
9. Cek tampilan field
10. Login & cek sidebar

---

## 📊 Assertion
* Validasi URL
* Pesan error
* Visibility element
